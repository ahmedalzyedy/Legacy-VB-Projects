Legacy Visual Basic 6 Projects

مجموعة من الأنظمة البرمجية المطورة باستخدام Visual Basic 6، تهدف إلى استعراض مهارات بناء الأنظمة الإدارية، التعامل مع قواعد البيانات، وتصميم الواجهات البرمجية.

🛠 المشاريع المتضمنة:

 1. نظام إدارة الموظفين (Database Management)
- الوصف: نظام متكامل لإدارة بيانات الموظفين.
- المهارات المستخدمة: الربط مع قواعد بيانات Access عبر مكتبة DAO، تنفيذ عمليات (إضافة، حذف، بحث، تنقل بين السجلات).

 2. وحدة العمليات الحسابية والمنطقية (Math Logic)
- الوصف: محرك حسابي لإجراء العمليات المعقدة.
- المهارات المستخدمة: الحلقات التكرارية (Loops)، معالجة أنواع البيانات الكبيرة (Long & Double)، ومنطق البرمجة الإجرائية.

 3. واجهات المستخدم والقوائم (GUI)
- **الوصف:** تصميم واجهات تفاعلية تدعم القوائم المنسدلة والتحكم في عناصر النموذج.
- **المهارات المستخدمة:** تصميم Menu Editors، التعامل مع ListBoxes، وتحسين تجربة المستخدم (UX) في البيئة الكلاسيكية.

 📈 الهدف من المشروع:
توثيق المهارات الأساسية في هندسة البرمجيات والقدرة على التعامل مع الأنظمة "الإرثية" (Legacy Systems) وتطوير منطق برمجى سليم وقابل للتطوير.
